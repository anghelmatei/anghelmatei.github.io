# anghelmatei.github.io

Personal portfolio website for **Matei Anghel**. This site showcases projects, experience, and a tech stack using a clean, responsive design.

## Features
- Dark / Light / Neutral theme with dynamic theme picker
- Custom animated cursor and smooth scroll effects
- Responsive layout for desktop and mobile
- Interactive hero section and timeline

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/anghelmatei/anghelmatei.github.io.git
   ```
2. Open `index.html` in a browser to view the site locally.

## Deployment
The site is hosted on GitHub Pages. Push changes to the `main` branch to automatically update the live site.
